# Massive.com AAPL Stock Pipeline & Power BI Dashboard

Built with Microsoft Fabric (trial) to:
- Fetch 30-day batches from Massive.com API
- Flatten nested JSON in PySpark notebooks
- Clean/dedupe/append data daily (growing history)
- Visualize in interactive Power BI report (candlestick trends, KPIs, slicers)
